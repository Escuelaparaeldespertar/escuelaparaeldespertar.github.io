Instrucciones de uso
Cómo conectar este panel con tu biblioteca
Paso 1 — Subir admin.html al servidor
Sube este archivo a la misma carpeta raíz donde está tu index.html

El archivo admin.html debe estar en la misma carpeta que index.html, flipbook.html y libros.js. Accede a él por URL directa, por ejemplo: https://tudominio.com/admin.html

Paso 0 — Sincronizar la biblioteca existente
Si ya tienes libros en libros.js, impórtalos primero desde el Dashboard

Al abrir el panel por primera vez verás una tarjeta amarilla en el Dashboard con el botón "Importar datos de libros.js". Al hacer clic, se cargan automáticamente todas las colecciones y libros existentes en el panel. Solo necesitas hacerlo una vez.

Paso 2 — Agregar colecciones y libros
Usa las secciones "Colecciones" y "Libros" del menú izquierdo

• Colecciones: Agrega el nombre, emoji e imagen del ícono.
• Libros: Completa título, autor, colección, URL del PDF y URL de la portada.
• Las URLs de portadas pueden ser rutas del servidor (Libros/categoria/portada.jpg) o URLs externas (Google Drive, Cloudflare R2, Dropbox, etc).
• Las URLs de PDFs funcionan igual — pueden ser R2, Google Drive o rutas locales del servidor.

Paso 3 — Exportar y subir libros.js
El paso más importante: actualizar el archivo de datos en el servidor

1. Ve a la sección Exportar datos del menú.
2. Haz clic en "Generar libros.js" y copia el código.
3. Abre el archivo libros.js en tu servidor (via FTP, cPanel File Manager, SSH, etc).
4. Reemplaza TODO el contenido con el código copiado.
5. Guarda. Los cambios se ven de inmediato en la biblioteca.

Respaldo de datos
Importante: los datos del panel se guardan en este navegador

Los datos se guardan en el localStorage del navegador. Si cambias de computadora o limpias el caché, los datos se perderían.

Recomendación: Usa la opción "Descargar respaldo .json" regularmente desde Exportar datos. Ese archivo lo puedes cargar en cualquier computadora con "Importar respaldo .json".
